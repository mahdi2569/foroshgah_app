# foroshgah_app
# برنامه مدیریت کالا با پایتون و Tkinter

این پروژه یک برنامه **مدیریت کالا** برای فروشگاه است که با زبان **پایتون** و کتابخانه **Tkinter** ساخته شده و از **SQLite** برای ذخیره داده‌ها استفاده می‌کند.

---

## امکانات برنامه

- افزودن کالا با مشخصات: نام، قیمت خرید، قیمت فروش و تعداد
- جستجوی کالا بر اساس نام
- حذف کالا از پایگاه داده
- ویرایش اطلاعات کالا
- ضرب تعداد در قیمت خرید برای محاسبه مقدار کل
- نمایش لیست کامل کالاها

---

## پیش‌نیازها

- Python 3.x
- Tkinter (به طور پیش‌فرض با پایتون نصب می‌شود)
- SQLite (به طور پیش‌فرض با پایتون نصب می‌شود)
